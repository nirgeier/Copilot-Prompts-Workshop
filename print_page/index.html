    Workshop Overview¶

    This 4-hour intensive workshop is designed to transform you from a GitHub Copilot user into a Copilot power user. You won’t write code manually‚Äîinstead, you’ll master the art of prompting to make Copilot write perfect code for you.

    What Makes This Workshop Unique

    • Zero manual coding - All changes through Copilot prompting only
    • Automated validation - Run tests to verify your prompts worked
    • Progressive difficulty - From trivial to advanced challenges
    • Real-world scenarios - Practical skills you’ll use daily

    What You’ll Master¶

    Core Skills¶

    • Inline Suggestions: Craft prompts that generate exact code you need
    • Chat Modes: Use Ask, Plan, and Agent modes effectively
    • Beast Mode: Leverage multi-file editing for complex refactors
    • Custom Instructions: Create project-wide coding standards
    • Custom Agents: Build specialized AI assistants

    Workshop Structure¶

    Phase 1: Foundations (30 min)

    Exercises 1-3: Basic operations (multiply, divide, power)

    Master inline suggestions with simple, focused prompts.

    Phase 2: Enhancement (45 min)

    Exercises 4-6: Modulo, square root, history tracking

    Learn to prompt for features with multiple components.

    Phase 3: Agent Mode (60 min)

    Exercises 7-9: Undo, export, constants

    Use Agent mode for complex multi-file changes.

    Phase 4: Custom Instructions (45 min)

    Exercises 10-11: Docstrings, type hints

    Apply project-wide standards automatically.

    Phase 5: Custom Agents (60 min)

    Exercises 12-13: Test generation, performance optimization

    Build and use specialized AI agents.

    Phase 6: Advanced (40 min)

    Exercises 14-15: Plugin system, GUI interface

    Master complex architectural prompting.

    üõ†Ô∏è Troubleshooting¶

    Copilot Not Working¶

    Issue: No suggestions appearing
    Solutions: - Check status bar (bottom-right) - should say “Copilot: Active” - Reload VS Code: Cmd+Shift+P ‚Üí “Reload Window” - Verify subscription: Visit https://github.com/settings/copilot

    Issue: Wrong suggestions
    Solutions: - Be more specific in comments/function names - Try Chat Ask instead: Ctrl+Shift+I (or Cmd+Shift+I on Mac) - Provide more context about what you want

    Tests Failing¶

    Issue: pytest tests/test_ex1.py shows failures
    Solutions: 1. Read the error message carefully 2. Ask Copilot: “Why is test_multiply_basic failing?” 3. Check if function exists with correct name 4. Verify function signature matches expected types

    Issue: Import errors
    Solutions: 

    # Reinstall dependencies
pip install -r requirements.txt

# Check Python version
python --version  # Should be 3.12+

    Setup Issues¶

    Issue: pytest: command not found
    Solution: 

    pip install pytest
# or
pip install -r requirements.txt

    Issue: Wrong Python version
    Solution: 

    # Use Python 3.12+ specifically
python3.12 -m pip install -r requirements.txt
python3.12 -m pytest tests/test_ex1.py

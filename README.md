## Workshop Overview

**Duration**: 4 hours  
**Focus**: Prompting skills (not coding skills)  
**Target Audience**: Developers/QA/DevOps with mixed backgrounds  
**Prerequisites**: No Python knowledge, VS Code, GitHub Copilot subscription

---

### What You'll Learn

- All GitHub Copilot modes (Inline, Chat Ask/Plan/Agent, Beast Mode)
- When to use each mode for maximum effectiveness
- Craft effective prompts for different scenarios
- Configure custom instructions and agents (`.github` directory)
- AI Driven development
- Test-driven development with AI
- Project-wide AI behavior configuration
